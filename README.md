# Finly

**Finly** is a modern and fully functional financial and accounting management system. Built with **Angular 21**, the application provides comprehensive control over a company's financial health by offering tools to monitor revenue, expenses, and client relationships.

## üöÄ Key Features

* **Interactive Dashboard**: Provides a real-time overview of key performance indicators, including total revenue (125,430 z≈Ç), monthly expenses (48,920 z≈Ç), unpaid invoices (23), and active client counts (48).
* **Invoice Management**: Features a complete list of documents with a status system (Paid, Pending, Overdue), payment deadlines, and a detailed summary of the total value to be paid (123,155,403.08 z≈Ç).
* **Expense Tracking**: Includes a detailed cost registry with automatic calculation of "Top 3" categories: Rent (4,500 z≈Ç), Services (3,500 z≈Ç), and Software (2,702 z≈Ç). It also identifies specific vendors and payment methods.
* **Client Database (CRM)**: A module for managing contractors (e.g., ABC Sp. z o.o., Tech Solutions). It stores contact details, NIP (Tax ID), and the total revenue history generated by each client.
* **Advanced Reporting**: Visualizes net profit trends, bar chart comparisons of revenue versus expenses, and a breakdown of revenue by category, such as IT Services (45%) and Consulting (28%).
* **Tax Summaries**: Automatically calculates VAT due (28,809 z≈Ç) and VAT claimed (11,292 z≈Ç). It also determines the total VAT payable (17,517 z≈Ç) along with the payment deadline.

## üíª Installation and Setup

To run the project in a local environment:

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Start the application**:
    ```bash
    npm start
    ```
    The application will be available at `http://localhost:4200`.
3.  **Build for production**:
    ```bash
    npm run build
    ```

## üìã Available Scripts

## üíæ Data Storage

The application is fully interactive, and all entered data is persistently saved in the browser's **localStorage**. This ensures that information regarding invoices, expenses, and clients remains available even after a page refresh, allowing for a seamless experience without the need for an external database configuration.
